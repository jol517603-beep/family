How to play

~ 4 simple moves ~
  1. Pick a story card (A–T) and read it aloud.
  2. Find the matching identity card (1–20).
  3. Paste the identity card in the correct generation band.
  4. Draw the relationship line using the legend below.

Line legend

~ how to draw the relationships ~
  • marriage · solid rose line
  • parent → child · solid jade line
  • siblings · dashed tan line
  • adopted / kept · dotted gold line
  • secret line · pencil only
